## Voting app


### Backend
- **Platform:** Node.js with Express.js
- **Database**: PostgreSQL with knex.js
- **Authentication**: Email, Twitter, Facebook, Github, Google
- **Deployment**: Heroku

### Frontend
- **Framework** React.js with Redux
- **CSS Preprocessor** Sass
- **HTML Preprocessor** Jade/Pug
